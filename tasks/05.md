## Atenção - Erros a Evitar

- O campo `conclusionRate` retornado pela API é um decimal entre 0 e 1 (ex: 0.66). Multiplique por 100 e arredonde antes de exibir como porcentagem (ex: `Math.round(value * 100)` → `66%`).
- No heatmap de consistência, **NÃO** diferencie visualmente datas futuras de datas passadas sem atividade. Todas as células sem dados de treino devem usar o mesmo estilo (`border border-border`), independentemente de serem passadas ou futuras.
- O heatmap de consistência deve exibir os **últimos 3 meses** (mês atual + 2 meses anteriores), usando semanas reais do calendário (segunda a domingo) como colunas, agrupadas por mês.
